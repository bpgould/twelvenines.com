## About

Twelve Nines is an SRE, DevOps, and Platform Engineering blog. It’s also a personal repository of the tools, patterns, and lessons I’ve picked up throughout my career. All opinions expressed here are my own and do not represent the views of my employer, past or present.

## Open Source

This project is open source and available on GitHub. Feel free to use it as a template for your own static site. For license and attribution details, see the LICENSE and README files.

## AI Content Disclaimer

AI tools may be used during the writing and development process, but — as with any production system — I review all code and content before publishing. All final writing is human-authored.
