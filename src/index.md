This site aspires to 99.9999999999% uptime - that's twelve nines of reliability.

## About

This is a static website built with [Cobalt](https://cobalt-org.github.io/), a Rust-based static site generator. The site is hosted on GitHub Pages and deployed automatically via GitHub Actions.

## Getting Started

To work with this site locally:

1. Install Cobalt: `cargo install cobalt-bin`
2. Serve the site: `cd src && cobalt serve`
3. Build the site: `cd src && cobalt build`

See [SETUP.md](https://github.com/bgould/twelvenines.com/blob/main/SETUP.md) for more details.
